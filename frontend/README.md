# AVISHU Superapp Frontend

Flutter-приложение для клиента, франчайзи и производства.

## Что есть

- вход по ролям
- каталог и корзина
- оформление и оплата заказа
- заказы клиента с этапами
- рабочее место франчайзи
- очередь и команда производства
- быстрые действия для сотрудника

## Запуск

Из корня проекта:

```bash
./scripts/run_demo.sh
```

Ручной запуск:

```bash
cd frontend
flutter pub get
flutter run --dart-define=API_BASE_URL=http://10.0.2.2:8000
```

## Основной сценарий

1. Клиент создает заказ и оплачивает его.
2. Франчайзи подтверждает заказ и передает его в производство.
3. Менеджер распределяет задачу.
4. Сотрудник берет этап в работу и завершает его.
5. Клиент видит готовность заказа.
